# Примеры использования и тестирование

Эта папка содержит примеры использования библиотеки и скрипты для тестирования.

## Файлы:

### 1. `basic_usage.py` - Базовые примеры
Демонстрирует основные возможности библиотеки:
- Создание дискретных случайных величин
- Работа с монетой и кубиком
- Операции со случайными величинами (сложение, умножение)
- Умножение на скаляр
- Сохранение и загрузка

**Запуск:**
```bash
python examples/basic_usage.py
```

### 2. `advanced_operations.py` - Продвинутые примеры
Демонстрирует более сложные операции:
- Умножение двух случайных величин
- Постепенное добавление значений
- Биномиальное распределение
- Асимметрия и эксцесс
- Цепочки операций

**Запуск:**
```bash
python examples/advanced_operations.py
```

### 3. `test_load_files.py` - Тестирование загрузки JSON файлов
Автоматически тестирует загрузку всех JSON файлов из папки `test_data`:
- Проверяет корректность загрузки
- Выводит распределение и статистику
- Проверяет, что сумма вероятностей = 1
- Выводит итоговый отчет

**Запуск:**
```bash
python examples/test_load_files.py
```

## Тестовые данные

В папке `test_data/` находятся примеры JSON файлов для загрузки в GUI приложение:

- `dice.json` - Бросание игрального кубика
- `coin_toss.json` - Подбрасывание монеты
- `binomial_5_03.json` - Биномиальное распределение
- `custom_distribution.json` - Произвольное распределение
- `two_dice_sum.json` - Сумма двух кубиков
- `poisson_like.json` - Распределение, напоминающее Пуассоновское

## Как использовать тестовые файлы в GUI:

1. Запустите приложение:
   ```bash
   python main.py
   ```

2. В меню выберите **"Файл" → "Загрузить"**

3. Перейдите в папку `test_data/`

4. Выберите любой JSON файл для загрузки

5. Проверьте:
   - Графики (закон распределения, полилайн, функция распределения)
   - Статистические характеристики (меню "Операции" → "Статистика")
   - Редактирование значений в таблице
   - Сохранение изменений

## Тестирование работоспособности:

Для полного тестирования работоспособности:

1. Запустите примеры использования:
   ```bash
   python examples/basic_usage.py
   python examples/advanced_operations.py
   ```

2. Запустите тест загрузки файлов:
   ```bash
   python examples/test_load_files.py
   ```

3. Загрузите файлы в GUI приложение и проверьте:
   - Корректность отображения графиков
   - Правильность расчета статистики
   - Возможность редактирования
   - Сохранение и загрузку
