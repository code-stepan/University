# Тестовые данные для приложения

Эта папка содержит примеры JSON файлов с дискретными случайными величинами для тестирования приложения.

## Файлы:

1. **dice.json** - Бросание игрального кубика (равномерное распределение от 1 до 6)

2. **coin_toss.json** - Подбрасывание монеты (0 = решка, 1 = орел)

3. **binomial_5_03.json** - Биномиальное распределение B(5, 0.3)

4. **custom_distribution.json** - Произвольное распределение (4 значения)

5. **two_dice_sum.json** - Сумма двух кубиков (распределение от 2 до 12)

6. **poisson_like.json** - Распределение, напоминающее Пуассоновское

## Использование:

1. Запустите приложение: `python main.py`
2. В меню выберите "Файл" → "Загрузить"
3. Выберите любой JSON файл из этой папки
4. Проверьте графики, статистику и функциональность

## Формат файлов:

Все файлы следуют формату:
```json
{
  "pmf": [
    [значение1, вероятность1],
    [значение2, вероятность2],
    ...
  ]
}
```

где `pmf` (probability mass function) - функция вероятности массы.
